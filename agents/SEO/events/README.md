# SEO events (optional)

Timed campaigns. Prefer tying each event to **one site**.

In `BRIEF.md` set:

```md
| **Site slug** | `shoprally` |   # must match agents/SEO/sites/{slug}
```

Do not use events to store the shared playbook — that lives in `model/`.  
Do not mix multiple businesses’ campaign data in one event folder.

Template: [`_template/`](./_template/).
