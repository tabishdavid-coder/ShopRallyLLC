# Module 05 — Measurement

## Required

- [ ] Google Search Console property verified for the live host
- [ ] Analytics (GA4 or agreed tool) on all indexable templates
- [ ] Baseline snapshot date recorded in site DEPLOYMENT.md
- [ ] KPI set chosen from PROFILE (e.g. organic demo clicks, ranking set, impressions)

## Cadence

| When | What |
|------|------|
| Weekly (active deploy) | Index coverage + new URL status |
| Monthly | Query/page movers; content gaps |
| After model bump | Diff site vs new model version |

Never share another site’s GSC or analytics data across `sites/*` folders.
