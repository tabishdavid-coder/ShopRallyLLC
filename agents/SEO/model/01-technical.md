# Module 01 — Technical SEO

Apply to every site unless PROFILE says “no index / staging only.”

## Requirements

- [ ] Canonical host (www vs apex) + HTTPS redirects
- [ ] `robots.txt` allows public marketing URLs; blocks private/app paths
- [ ] XML sitemap of indexable URLs; linked from robots or known to GSC
- [ ] No soft-404s; real 404 for missing pages
- [ ] Unique indexable URLs (no duplicate parameter variants)
- [ ] Mobile-friendly layout; LCP/CLS/INP within acceptable range for key templates
- [ ] OG/Twitter tags resolve absolute image URLs
- [ ] Favicon + basic brand assets load without errors

## Deliverables (per site)

Record evidence in site `DEPLOYMENT.md` → Technical (URLs of sitemap, robots, GSC property).
