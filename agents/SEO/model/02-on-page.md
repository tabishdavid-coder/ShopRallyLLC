# Module 02 — On-page SEO

## Per indexable page

- [ ] Unique `<title>` (intent + brand; avoid stuffing)
- [ ] Unique meta description (benefit + CTA tone)
- [ ] Single clear `H1`
- [ ] Logical H2/H3 outline
- [ ] Primary keyword / topic stated once in intro (natural language)
- [ ] Internal links to 2–5 related pages
- [ ] Image alt text where images convey meaning
- [ ] Schema where it fits (Organization, WebSite, FAQ, Product/Software, LocalBusiness — pick from PROFILE)

## Sitewide

- [ ] Nav + footer link to money pages (pricing, features, contact/demo)
- [ ] Consistent brand name as in PROFILE
- [ ] No thin doorway pages

## Templates

Copy page inventory into site `custom/page-inventory.md` during deploy.
