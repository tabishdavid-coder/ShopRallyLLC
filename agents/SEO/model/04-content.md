# Module 04 — Content SEO

## Pillars (define per site in PROFILE / custom)

Typical SaaS / shop-software pattern:

1. **Category** — what the product is
2. **Jobs-to-be-done** — problems (estimates, job board, payments…)
3. **Comparisons / alternatives** — only with honest, non-infringing copy
4. **Resources** — guides that earn links and support sales

## Requirements

- [ ] Keyword → URL map (one primary intent per URL)
- [ ] No cannibalizing titles across pages
- [ ] Publish cadence noted in site BUILD-STATE
- [ ] Refresh stale pages quarterly (or as PROFILE sets)

## Do not

Copy competitor trade dress or trademarked product UI claims as your own.
