Direct answer: Is Vercel needed for every shop?

No. Vercel is one shared host for the whole company — not per shop.

ShopRally is multi-tenant SaaS (Software as a Service): one Next.js deployment on Vercel, one Neon Postgres database with rows tagged by shopId, one Clerk auth app. Each repair shop is a tenant login — not a new server.

Cost rises with shops only because usage rises (requests, storage, email, AI seats) — you do not buy a new Vercel project for shop #2, #50, or #200. Analogy: one kitchen, many customers — not a kitchen per customer.

1. The story in one page

ShopRally Ignition wins independents who need a full shop CRM without paying for MOTOR labor data, SMS (Short Message Service) marketing, or Stripe Connect on day one. Upsell is ShopRally AI Plus ($20/mo): Smart / freeform Repair Order (RO) intake, labor assist, and the advisor mobile app.

  • Infrastructure = single shared stack (Vercel + Neon + Clerk …)
  • Team = solo founder — hiring is optional reinvestment, not a survival cost
  • Gross story @ 200 ≈ $14.3k MRR · $172k ARR · ~$1.9k vendors · ~87% contribution after vendors
Investor-relevant unit: software margin before founder living expenses. Infra is not the bottleneck — distribution and AI Plus attach are.

5. Security & mobile (shared, not per shop)

Security cash vs build

  • TLS / HTTPS — $0 (included on Vercel)
  • Auth / org boundaries — inside Clerk bill
  • Data isolation — every query scoped by shopId
  • Sentry / uptime — ~$0 → ~$120/mo as you grow
  • Neon PITR (Point-In-Time Recovery) — inside DB plan
  • SOC 2 path — later (~$6k–$18k/yr), not day-one for 25–50 shops
  • Security vendor per shop — none

Mobile (AI Plus economics)

  • Customer pays AI Plus $20/mo (intake + labor + advisor app)
  • Store + Expo/EAS + push (APNs / FCM) ≈ $9 → $55/mo platform
  • Gemini LLM (Large Language Model) ≈ $2.50–$3.80 per AI seat/mo
  • One App Store / Play listing — shops share the same binary + API
  • No separate mobile host per shop

6. Why this is investable without hiring

  • Gross margin after vendors is software-grade (~85%+ at 200)

  • Variable costs track attach — AI & Stripe fees rise when customers pay

  • Hiring becomes a reinvestment choice after contribution is real
